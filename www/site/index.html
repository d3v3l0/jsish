Jsi is a small javascript interpreter written in C with builtin Database and Websocket support.

What distinguishes it from other script languages is:

  • Typed functions
  • Embedded access to C struct data
  • Executable is an atomic standalone incorporating it's own zip filesystem
  • No need to learn yet-another-syntax: it's javascript

Jsi was designed by PDQI, specifically for industrial embedded applications requiring diagnostics, testing, configuration, and data-acquisition capabilities
